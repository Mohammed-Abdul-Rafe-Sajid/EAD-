what is mern stack?

MERN Stack is the abbreviation for MongoDb, ExpressJS, React, NodeJS
it is used for Webdevelopment , We can define it as the collection of JAVASCRIPT based technologies used to build web applications
, its the full stack framework which allows the useres to create both the front end and the backend

The technologies are
  • Mongodb
  • ExpressJS
  • React
  • Node.js
